# simapp
CLI text similarity calculator

# Download
wget https://github.com/MrDznonyme/simapp/releases/download/v2/Simapp_v2.zip

# Extract
**Linux**

apt install unzip

unzip Simapp_v2.zip

# Usage
gcc simapp.cpp -o simapp -lstdc++ -lm

./simapp file1 file2

arguments:

-s : print all words frequent
